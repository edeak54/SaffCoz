# SaffCoz?
A simple and quick website I made for a fragance company, Saffcoz. Made specifically for the final assignment of a course in my university.

# How to run:

- Open VS Code.
- Make sure you have Live Server extension, if not you can find it here https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer.
- Clone this repository.
- Open the code folder in VS Code.
- On any html code, right click and click "Open with Live Server".
- The website will open automatically on your browser.
